Using the T-discover Tool

The T-discover tool helps you analyze and group reports based on their similarity. Follow the steps below to use the tool effectively:

  1. Prepare the Input: Create a CSV file with two columns - "Report Name" and "Objects Used". The "Objects Used" column should list the tables, columns, KPIs, and dimensions used in each report.
  2. Upload the CSV File: Open the T-discover tool and upload the CSV file using the provided file input.
  3. Define Similarity Buckets: On the homepage, set the ranges or buckets for similarity scores. For example, you can choose a bucket size of 70-100% similarity.
  4. Generate Results: Click on the "Generate" button to analyze the reports and calculate the similarity scores.
  5. View Grouped Reports: The tool will sort the reports into different buckets based on their similarity scores. In each bucket, the tool will create groups of reports that are similar to each other. You can view the grouped reports in the output table.
